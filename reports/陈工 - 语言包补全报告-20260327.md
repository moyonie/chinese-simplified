- `flarum-mentions` (提及扩展)
- `flarum-likes` (点赞扩展)
- 等等...

### 2.2 引用语法规范

✅ 所有跨文件引用使用正确格式：
```yaml
# 正确格式
submit_button: => core.ref.save_changes
tags_label: => flarum-tags.ref.tags

# 错误格式（已避免）
submit_button: core.ref.save_changes
submit_button: =>core.ref.save_changes
submit_button: => ref.save_changes
```

### 2.3 复用翻译引用

✅ 正确使用 `ref` 命名空间复用翻译：

**core.ref 引用**:
- `=> core.ref.save_changes` (保存更改)
- `=> core.ref.color` (颜色)
- `=> core.ref.icon` (图标)
- `=> core.ref.okay` (确定)
- `=> core.ref.load_more` (加载更多)
- `=> core.ref.some_others` (其他用户)
- `=> core.ref.you` (您)
- `=> core.ref.reply` (回复)

**扩展 ref 引用**:
- `=> flarum-tags.ref.tags` (标签)
- `=> flarum-tags.ref.choose_tags` (选择标签)
- `=> flarum-flags.ref.flag_post` (举报帖子)
- `=> flarum-sticky.ref.sticky` (置顶)
- `=> flarum-subscriptions.ref.follow` (关注)
