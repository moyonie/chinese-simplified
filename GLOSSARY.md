## ⚠️ 易混淆术语对照

| 英文 | 正确翻译 | 错误翻译 | 说明 |
|------|---------|---------|------|
| conversation | 对话 | 会话 | 私信场景用"对话" |
| dialog | 对话框 | 对话 | UI 元素用"对话框" |
| message | 私信 | 消息 | messages 模块统一用"私信" |
| notification | 通知 | 消息 | 系统通知用"通知" |
| repository | 软件源 | 仓库 | extension-manager 用"软件源" |
| post | 帖子 | 文章/帖子 | 统一用"帖子" |
| discussion | 讨论 | 话题/讨论 | 统一用"讨论" |
| tag | 标签 | 标签/标记 | 统一用"标签" |
| flag | 举报 | 标记 | 举报功能用"举报" |
| sticky | 置顶 | 粘滞 | 论坛术语用"置顶" |
| suspend | 暂停 | 挂起/暂停 | 用户管理用"暂停" |
| extension | 扩展 | 插件 | Flarum 官方术语用"扩展" |
